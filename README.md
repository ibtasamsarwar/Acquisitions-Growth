# Acquisitions Growth React Website

A responsive React + Vite website built from the provided HTML designs.

## Pages

- `/` Home page
- `/contact` Contact page

## Tech

- React
- Vite
- React Router

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```text
src/
  components/
    ContactForm.jsx
  pages/
    HomePage.jsx
    ContactPage.jsx
  App.jsx
  main.jsx
```

## Assets

- `logo.png` for site branding
- `favicon.png` for browser tab icon
